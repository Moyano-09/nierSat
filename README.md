# Nier Automata Space Portfolio Scaffold

This repository is intentionally structured as a learning canvas instead of a finished app.

## How To Run

From the project root:

```powershell
npm install
cd Frontend
npm run dev
```

Or from the project root without changing directories:

```powershell
npm run dev
```

Default local URL:

- frontend app: `http://localhost:5173`

## Current Direction

- `Frontend/` is the active Vue 3 + Vite application.
- `Frontend/src/main.ts` is the frontend bootstrap entry point.
- `backend/` is reserved for a future C# API.

## Root Scripts

These root scripts proxy into `Frontend/`:

- `npm run dev`
- `npm run build`
- `npm run preview`
- `npm run type-check`
