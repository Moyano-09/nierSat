YoRHa Space Archive

A portfolio-shaped interface for collecting and displaying imagery, mission data, and scientific records from NASA, ESA, and other public space sources.

Right now this is a scaffold. The layout, naming, and file boundaries are here so you can learn how a real full-stack version would grow.

How The Pieces Talk

1. Frontend

Requests data from your own backend instead of calling every third-party API directly in the browser.

2. Backend

Normalizes NASA, ESA, or other provider responses into one shape your frontend can trust.

3. Database

Stores cache, saved items, and portfolio content so the site becomes yours instead of just a temporary API viewer.
